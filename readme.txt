WP Geo Tags fuer Deutschland

== Description ==

Mit WP Geo Tags for Germany koennen die Meta Tags geo.region, geo.placename, geo.position and ICBM fuer jeden einzelnen Artikel und jede einzelne Seite separat fuer Deutschland angegeben werden. Support: [http://www.risikolebensversicherungen.ag/](http://www.risikolebensversicherungen.ag/)

== Installation ==

1. Extract `wp-geo-tags-for-germany.1.0.zip`.
2. Upload the `wp-geo-tags-for-germany`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
